# 渲染方程笔记：数学形式与物理直觉

## 1. 渲染方程的统一形式

渲染方程是计算机图形学中描述**光能在场景中传播与散射**的统一积分方程：

$$
I(x, x') = g(x, x') \left[ \epsilon(x, x') + \int_S \rho(x, x', x'') I(x', x'') \, dx'' \right]
$$

---

## 2. 各项含义（数学 → 物理）

| 符号                 | 数学含义                                    | 物理直觉                          |
| -------------------- | ------------------------------------------- | --------------------------------- |
| $ I(x, x') $         | 从点 $x'$ 到点 $x$ 的**未遮挡两点传输强度** | 光线从一点到另一点的“亮度”        |
| $ g(x, x') $         | 几何项：$1/r^2$ 或 0（遮挡）                | 距离衰减 + 可见性（有无遮挡）     |
| $ \epsilon(x, x') $  | 从 $x'$ 发出的**自发光强度**                | 光源本身的发光                    |
| $ \rho(x, x', x'') $ | 三点散射函数（BRDF 的推广）                 | 表面如何反射来自 $x''$ 的光到 $x$ |
| $ S $                | 所有表面的并集                              | 场景中所有物体表面                |
| 积分项               | 所有表面点对 $x'$ 的入射光贡献              | 全局光照的“反射链”                |

---

## 3. 物理直觉：光能守恒

> **渲染方程的本质**：  
> 从点 $x'$ 到点 $x$ 的光强 = 自发光 + 所有其他表面点反射来的光。

这就像说：  
**你看到的颜色 = 自己发的光 + 别人反射给你的光**。

---

## 4. 与经典辐射度方程的对比

| 特性     | 辐射度方程         | 渲染方程                   |
| -------- | ------------------ | -------------------------- |
| 表面假设 | 仅限**理想漫反射** | 任意BRDF                   |
| 角度依赖 | 无                 | 有                         |
| 可见性   | 需额外计算         | 由 $g$ 项统一处理          |
| 适用场景 | 室内、漫反射主导   | 全局光照、镜面、折射、焦散 |

---

## 5. 常见近似方法（从方程出发）

### 5.1 Neumann 级数展开

$$
I = g\epsilon + gM g\epsilon + gM gM g\epsilon + \cdots
$$

物理含义：  
- 第1项：直接光照  
- 第2项：一次反射  
- 第3项：两次反射  
- 依此类推 → **光线路径的“跳数”**

---

### 5.2 路径追踪（Path Tracing）

- 每次只追踪**一条路径**（而非射线树）
- 用 Monte Carlo 方法估计积分
- 优点：**不浪费计算在高阶反射上**

---

## 6. 重要物理现象的支持

| 现象                       | 方程中的来源             |
| -------------------------- | ------------------------ |
| 阴影                       | $g(x, x')$ 为 0          |
| 漫反射                     | $\rho$ 为常数            |
| 镜面反射/折射              | $\rho$ 为 delta 函数形式 |
| 颜色溢出（Color Bleeding） | 多次反射项               |
| 焦散（Caustics）           | 光通过折射表面聚焦       |
| 环境光照                   | 来自背景表面 $S_0$       |

---

## 7. 求解方法对比

| 方法                              | 本质                    | 优缺点                 |
| --------------------------------- | ----------------------- | ---------------------- |
| 辐射度法                          | 矩阵求解（漫反射）      | 适合低频光照，计算量大 |
| 光线追踪                          | 递归射线树              | 高阶反射成本高         |
| 路径追踪                          | Markov链 + Monte Carlo  | 无偏、通用、慢         |
| 分层采样（Hierarchical Sampling） | 自适应细分 + 重要性采样 | 方差小、收敛快         |

---

## 8. 总结一句话

> **渲染方程 = 光能守恒 + 全局光照的统一数学表达**，它统一了光线追踪、辐射度、路径追踪等算法，是现代真实感渲染的理论基石。

# Lecture 3

## 📌 一、核心概念

- **几何变换**：将物体从一个位置映射到另一个位置的操作。
- **目的**：
  - 允许在局部坐标系中定义物体，再变换到世界坐标系。
  - 支持**实例化**（Instancing）：重复使用同一几何描述，节省内存。
- **变换对象**：点、多边形（通过变换顶点）、图像等。

---

## 📌 二、2D 基本变换（代数形式）

| 变换 | 公式                                                         |
| ---- | ------------------------------------------------------------ |
| 平移 | $ x' = x + t_x,\; y' = y + t_y $                           |
| 缩放 | $ x' = s_x \cdot x,\; y' = s_y \cdot y $                   |
| 剪切 | $ x' = x + sh_x \cdot y,\; y' = y + sh_y \cdot x $         |
| 旋转 | $ x' = x \cos\theta - y \sin\theta,\; y' = x \sin\theta + y \cos\theta $ |

> 注：旋转为正**逆时针**，遵循右手定则；2D 旋转可交换，但 3D 不可。

---

## 📌 三、矩阵表示（核心）

- **2D 线性变换**（缩放、旋转、剪切）可用 **2×2 矩阵**表示。
- **平移不是线性变换**，需用**齐次坐标**（3D 向量表示 2D 点）：
  - 点：$(x, y, w)$，实际位置为 $(x/w, y/w)$
  - 平移用 **3×3 矩阵**表示。

### 2D 变换矩阵（齐次坐标）：

| 变换 | 矩阵                                                         |
| ---- | ------------------------------------------------------------ |
| 平移 | $\begin{bmatrix}1&0&t_x\\0&1&t_y\\0&0&1\end{bmatrix}$      |
| 缩放 | $\begin{bmatrix}s_x&0&0\\0&s_y&0\\0&0&1\end{bmatrix}$      |
| 旋转 | $\begin{bmatrix}\cos\theta&-\sin\theta&0\\\sin\theta&\cos\theta&0\\0&0&1\end{bmatrix}$ |
| 剪切 | $\begin{bmatrix}1&sh_x&0\\sh_y&1&0\\0&0&1\end{bmatrix}$    |

---

## 📌 四、变换类型层级

- **线性变换**：缩放、旋转、剪切（原点不变，保直线、保平行、保比例）。$$
- **仿射变换**：线性变换 + 平移（不保原点，保直线、保平行、保比例）。
- **投影变换**：仿射变换 + 投影扭曲（不保平行，保直线，保“交比”）。

---

## 📌 五、矩阵组合（重要）

- 变换通过**矩阵乘法**组合，顺序**不可交换**（除非特殊情况）。
- 组合顺序：先应用的变换在**右边**，后应用的在**左边**。
- 示例：绕任意点 $(a,b)$ 旋转：
  1. 平移 $(-a,-b)$ 到原点
  2. 旋转
  3. 平移 $(a,b)$ 回去
- 逆变换：
  - 平移：取反位移
  - 旋转：转置矩阵（即逆）
  - 缩放：取对角线倒数

---

## 📌 六、3D 变换

- 使用 **4×4 齐次矩阵**（坐标 $(x,y,z,w)$）
- **平移、缩放、剪切**与 2D 类似，扩展到三维。
- **3D 旋转**更复杂：
  - 需指定**旋转轴**和**角度**
  - 不满足交换律
  - 轴对齐旋转（绕 X/Y/Z）可用基本旋转矩阵
  - 任意旋转可由**欧拉角**组合，但存在**万向锁**问题
  - 更优方案：**四元数**或**指数映射**（讲义未展开）

---

## 📌 七、变换层次结构（场景图）

- 场景中坐标系呈**树状层次**。
- 每个节点存储相对于父节点的变换矩阵。
- 应用示例：
  - **机器人手臂**（关节层级）
  - **角色动画**（人体骨骼）
  - **多次实例化同一物体**（不同位置/缩放/旋转）
